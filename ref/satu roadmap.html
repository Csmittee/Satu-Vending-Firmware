Company Structure

PDPA Compliance
Donor consent flow is partially coded but incomplete. At national scale, PDPA violations carry criminal liability. Need legal review before launch.
Security Hardening
At national scale, you become a target. rateLimit.js is broken. Need Cloudflare WAF rules, D1 backup strategy, incident response plan.
IP Protection
File patent/utility model for the machine concept before public demo with partners. Trade secret for firmware (flash encryption already designed in Security_Protocol.txt).
Financial Audit Trail
Temple donations require transparent accounting for trust. Admin dashboard needs export, reconciliation reports, and eventually integration with Thai Revenue Department systems.
Team / AI Resource Plan
